# Magasin
French for shop! E-commerce wesbite for all kinds of knicks and knacks


## Setup

In order to fully install all packages & dependencies:
`npm install` in your run time environment

`npm run start-dev` and start shopping!

## Technologies Used

*React 
*Redux 
*Bootstrap
*Sequelize 
*Node.js 
*Express 
*Heroku
